# MVP_ENGENHARIA_DE_DADOS_WINE
Pipeline completo de Engenharia de Dados desenvolvido no Databricks utilizando o Wine Dataset (UCI). O projeto aplica a arquitetura Medallion (Bronze, Silver e Gold) para ingestão, tratamento e análise dos dados, incluindo consultas SQL, visualizações em Python e respostas a 15 perguntas analíticas sobre as características químicas dos vinhos.
